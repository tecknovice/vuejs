# install nodeJS v8.11.2
# install git
# git clone 
```
git clone https://github.com/tecknovice/vuejs
cd vuejs/todo
```
# install @vue/cli client package
```
npm install --global @vue/cli
```
# install bootstrap
```
npm install bootstrap@4.0.0
```
# run todo app
```
npm run serve
```
# access todo app
```
http://localhost:8080
```
